## 📘 Overview

The **Nextflow Training Week** is an intensive, community-supported training program designed to teach **workflow management and reproducible computational pipelines** using **Nextflow**.

The training combined:

* Instructor-led video tutorials
* Hands-on coding exercises
* GitHub Codespaces-based development environments
* Community forum discussions
* Live Zoom sessions and office hours

Throughout the training, participants developed and executed **realistic bioinformatics workflows** and were required to **demonstrate full conceptual and practical understanding** by completing an official **end-of-week assessment quiz**.

> **Passing the training required answering all assessment questions correctly**, ensuring mastery of core Nextflow concepts, workflow logic, and reproducible pipeline execution.

## 🎯 Relevance to My Field

As an MSc candidate in **Biochemistry & Molecular Biology**, with focused interests in **Molecular Cancer Biology, RNA Biology, and Bioinformatics**, this training provided:

* Translate **manual RNA-seq analyses** into **scalable, reproducible pipelines**
* Understand workflow execution and debugging for **high-throughput data**
* Develop **best practices for reproducible bioinformatics**
* Apply **cloud-based and containerized environments** for genomics workflows

## 🛠️ Skills Acquired

* Designing **declarative, modular workflows** with Nextflow DSL2
* Implementing **process-level logic and workflow orchestration**
* Managing **parallel execution and channel-based data flows**
* Building **multi-step, multi-sample RNA-seq pipelines**
* Aggregating outputs safely with `collect()` and `mix()`
* Containerized execution with **Docker** for reproducibility
* Using **GitHub Codespaces** for cloud-based workflow development
* Handling **sample metadata (CSV)** and multiple RNA-seq library types
* Running QC, trimming, alignment, and MultiQC reporting in RNA-seq
* Ensuring workflow **scalability, reproducibility, and maintainability**
