## 📝 Overview

This folder documents my **hands-on training, collaborative hackathon experience, and bioinformatics workflow development** from the **Marine Bioinformatics Hackathon**.

It includes:

1. **Structured Training Notes** — Documenting lectures, hands-on sessions, and 

   * Marine biodiversity genomics and conservation
   * Metagenomics analysis of microbial communities
   * 16S rRNA amplicon sequencing workflows
   * Bioinformatics analysis using the Galaxy platform
   * Taxonomic classification and visualization of microbial communities
   * Workflow automation and reproducible pipelines using Nextflow
   * Exploration of nf-core standardized bioinformatics pipelines

2. **Tutorial Bioinformatics Workflow** — Detailed **16S rRNA amplicon analysis workflow** using Galaxy and reproducible pipelines, covering:

   * Sequence import and quality control
   * OTU clustering and taxonomic classification
   * Visualization using Krona and phyloseq
   * Workflow documentation for reproducibility

3. **Hackathon Project Implementation** — Collaborative analysis of **Nile River freshwater samples**, including:

   * Seasonal microbial diversity analysis (Winter vs Summer)
   * OTU-based abundance charts and Krona visualizations
   * Full project report and presentation documentation
